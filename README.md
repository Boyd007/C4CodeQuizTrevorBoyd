# # C4 Code Question Quiz

# Purpose:
<img width="744" alt="image" src="https://user-images.githubusercontent.com/105176171/177671737-b518d187-19d0-4726-9017-532a88e06a24.png">

This application is designed to test a students skills in javascript.  The quiz is timed and players are penalized chronologically for every wrong answer. The player can then save their high score.  This is all in an effort to interview better for developer positions.
